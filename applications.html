Restaurant Maja


Restaurant Maja is a React Native powered mobile application hosted through expo that helps users choose what they are in the move for by scrolling through a data.json powered listed of pics. Once the user selects a food that they would like to eat, the application extracts the users GPS location and pulls up the nearest restaurants that sell that food via Google Maps.
